# Blog Application

A full-stack blog application with React frontend and Spring Boot backend.

## Features

- User authentication (register/login)
- Create, read, update, delete blog posts
- Responsive design
- JWT authentication
- Pagination

## Technologies

**Frontend**:
- React 18
- Vite
- React Router 6
- Axios
- Tailwind CSS

**Backend**:
- Spring Boot 3.1
- Spring Security
- JWT Authentication
- MySQL
- JPA/Hibernate
